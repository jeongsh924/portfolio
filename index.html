# 👋 Welcome

Visual Studio Code UI를 모티브로 제작된 웹 퍼블리셔의 포트폴리오입니다.

Start

  • `about.html` : 자기소개
  • `skills.css` : 기술 스택
  • `projects.js` : 프로젝트 경험
  • `logs.md` : 경력 및 개발 회고
  • `contact.json` : 연락 수단

Options

  • `user` : 정보 요약
  • `setting : 테마 변경`

> 웹 퍼블리셔 실무 경험을 바탕으로 하여 한 눈에 파악하기 좋은 구성을 만들도록 노력하였습니다.
