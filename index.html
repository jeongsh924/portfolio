# 🎈 User

7년 10개월의 경력을 가진 웹 퍼블리셔

한 기업에서 7년 10개월 동안 단독으로 퍼블리싱 및 유지보수를 맡으며, 반복적인 작업을 구조화하고 템플릿화하여 생산성과 품질을 동시에 개선했습니다. 이 과정에서 SCSS와 jQuery 기반의 퍼블리싱 자동화 및 반응형 웹 구축 역량을 키웠습니다.

퍼블리셔로서의 주도성과 실행력은 조직 내에서도 인정받아, 디자인팀과 개발팀 사이에서 기술적 허브 역할을 수행했으며, 실제로 개발팀 편입 제안을 받은 바 있습니다. 이후 일본에서 임베디드 개발을 경험했지만, 정적이고 폐쇄적인 환경보다는 웹 기반 UI 구현과 프론트엔드 개발에서 더 큰 가능성을 느껴 다시금 웹 퍼블리싱에 집중하고 있습니다.

현재는 React 기반 컴포넌트 개발 및 동적 UI 구현에 관심을 가지고 있으며, 단순 퍼블리싱을 넘어 사용자 경험을 향상시키는 UI 퍼블리셔로의 성장을 지향하고 있습니다.
